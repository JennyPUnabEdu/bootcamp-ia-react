<a name="introducción"></a>
## 1. Introducción

**Bienvenido a la Plataforma Bootcamp 360.**

Esta es una herramienta web interactiva diseñada para enseñarte a utilizar Inteligencia Artificial Generativa (como ChatGPT y Google Gemini) de forma práctica y entretenida. Olvídate de las presentaciones aburridas; aquí podrás interactuar con simuladores, responder preguntas en tiempo real y ver comparativas dinámicas.

### ¿Qué puedo hacer aquí?
*   **Aprender Conceptos:** Mediante diapositivas interactivas.
*   **Simular Chats:** Practicar "Prompt Engineering" en un entorno seguro que imita a las IAs reales.
*   **Comparar Modelos:** Ver en tiempo real las diferencias entre ChatGPT y Gemini.
*   **Evaluarte:** Responder quizzes y obtener retroalimentación inmediata.
*   **Descargar Recursos:** Obtener resúmenes en PDF de cada módulo.

<a name="requisitos-y-acceso"></a>
## 2. Requisitos y Acceso

### Requisitos necesarios
*   **Dispositivo:** Se recomienda usar un Computador (PC/Mac) o Tablet para visualizar mejor los gráficos.
*   **Navegador:** Use versiones actualizadas de Google Chrome, Microsoft Edge, Firefox o Safari.
*   **Conexión:** Internet estable (especialmente al entrar por primera vez).

### Cómo entrar
1.  Abra su navegador web favorito.
2.  Escriba la siguiente dirección: `proyectos.icinf.icfunab.cl/curso-ia`
3.  ¡Listo! Verá la pantalla de bienvenida.

<a name="navegación-y-uso"></a>
## 3. Navegación y Uso

### 3.1 Pantalla de Inicio
Al entrar, verá el **Menú de Módulos**. Cada tarjeta representa una lección (ej. "Módulo 1: Introducción", "Módulo 2: Prompting").
*   Haga clic en **"Iniciar Módulo"** en la tarjeta que desee estudiar.

### 3.2 Controles Básicos
Dentro de una lección, la navegación es intuitiva:
*   **Avanzar:** Haga clic en la flecha derecha (`>`) o en el lado derecho de la pantalla.
*   **Retroceder:** Use la flecha izquierda (`<`).
*   **Salir:** Use el icono de "Casa" (Home) arriba a la derecha para volver al menú principal.
*   **Barra de Progreso:** Abajo verá cuánto le falta para terminar.

### 3.3 Tipos de Actividades
Durante el recorrido encontrará:
1.  **Slides Informativas:** Lea el contenido y vea las imágenes.
2.  **Simuladores de Chat:** Aparecerá una ventana parecida a ChatGPT. Escriba lo que se le pida o presione "Run" para ver cómo responde la IA automáticamente.
3.  **Comparativas (Versus):** Deslice el cursor sobre la pantalla para ver las diferencias entre dos modelos de IA.
4.  **Quizzes (Preguntas):** Seleccione la alternativa correcta. El sistema le dirá al instante si acertó.
5.  **Descargas:** Busque el botón "Descargar PDF" al final de los temas para guardar el resumen.

### 3.4 Flujo de Estudio Recomendado
1.  Ingrese a la plataforma.
2.  Elija un módulo (comience por el 1).
3.  Avance slide por slide completando todas las interacciones.
4.  No se salte los Quizzes, le ayudarán a verificar si entendió.
5.  Al llegar al final ("Slide de Cierre"), descargue el material y vuelva al inicio.

> **Consejo:** Para una experiencia más inmersiva, presione `F11` en su teclado para poner el navegador en **Pantalla Completa**.

<a name="solución-de-problemas-comunes"></a>
## 4. Solución de Problemas Comunes

| Problema | Solución Sugerida |
| :--- | :--- |
| **La pantalla se queda en blanco** | Presione `F5` o el botón de recargar del navegador. Verifique que tiene internet. |
| **No puedo avanzar a la siguiente slide** | Algunas slides requieren que complete una acción primero (responder una pregunta o probar el chat). Hágalo e intente avanzar de nuevo. |
| **Página "No encontrada" (Error 404)** | Si guardó una dirección antigua, es mejor que entre de nuevo a la portada: `proyectos.icinf.icfunab.cl/curso-ia` |
| **Se ve "cortado" en mi celular** | La plataforma está optimizada para PC y Tablet. Si usa celular, intente girarlo horizontalmente. |

<a name="glosario-básico"></a>
## 5. Glosario Básico

*   **Prompt:** Es la instrucción o texto que le escribimos a la IA para que haga algo.
*   **IA Generativa:** Tipo de inteligencia artificial capaz de crear contenido nuevo (texto, imágenes) en lugar de solo analizar datos.
*   **Simulación:** En esta plataforma, es una recreación segura de cómo funciona una IA, sin usar datos reales ni cuentas externas.
*   **SPA:** Tecnología web que hace que el sitio sea rápido y no parpadee al cambiar de página.
